还有一点需要注意，pod setup在执行时，会输出Setting up CocoaPods master repo，但是会等待比较久的时间。这步其实是 Cocoapods 在将它的信息下载到 ~/.cocoapods目录下，如果你等太久，可以试着 cd 到那个目录，用du -sh *来查看下载进度。
如果安装成功,你会看到:Setup completed。那么恭喜了。

Cocoapods 简单使用

cd 到工程根目录

Cocoapods老用户可能需要注意一下，Podfile的写法现在统一为：

vim 环境编辑完 ESC 退出输入模式，“：wq” 保存退出

pod install

完成后用新的工程文件打开工程，在import头文件时，如果没有智能提示，需要在 Xcode 里工程文件做一下配置

TARGETS -> Search Paths -> User Header Search Paths 中 写入 ${SRCROOT} 再将后面参数改为recursive
